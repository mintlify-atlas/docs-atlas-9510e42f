The FreeRTOS Kernel is a market-leading real-time operating system (RTOS) for microcontrollers and small microprocessors. It is freely available under the MIT license and provides the essential building blocks for reliable, deterministic embedded applications.

## Quick integration

Add FreeRTOS Kernel to your CMake project in two steps:

```cmake CMakeLists.txt
include(FetchContent)

FetchContent_Declare(
  freertos_kernel
  GIT_REPOSITORY https://github.com/FreeRTOS/FreeRTOS-Kernel.git
  GIT_TAG        V11.1.0
)

# Define the config library before calling MakeAvailable
add_library(freertos_config INTERFACE)
target_include_directories(freertos_config SYSTEM INTERFACE include)

set(FREERTOS_HEAP "4" CACHE STRING "" FORCE)
set(FREERTOS_PORT "GCC_ARM_CM4F" CACHE STRING "" FORCE)

FetchContent_MakeAvailable(freertos_kernel)
```

Then link your application:

```cmake
target_link_libraries(my_app PRIVATE freertos_kernel)
```

<Note>
Replace `GCC_ARM_CM4F` with the port that matches your target hardware. See the [porting overview](/porting/overview) for a full list of supported architectures.
</Note>
